

# CSS Basic Exercises

This folder contains the basic HTML and CSS exercises for the `alx_html_css` project. It's intended to demonstrate foundational CSS layout, selectors, and styling techniques.

Contents
 - `index.html` — Main practice page demonstrating basic layout and styles.
 - `tweets.html` — A small example page styled to look like simple tweet cards.
 - `base.css` — Base styles and resets used across the practice pages.
 - `styles.css` — Page-specific styles for `index.html` and `tweets.html`.

Notes
 - Files are intentionally small and focused for learning CSS fundamentals.
 - Edit the CSS files and refresh the browser to see changes immediately.
